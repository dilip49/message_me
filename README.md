# README
 
This is simple chatting application using ruby on rails action cable. This application shows basic usage of action cable in rails. For more details about action cable visit https://guides.rubyonrails.org/action_cable_overview.html.

Local Installation

If you'd like to run the example locally, make sure you have Ruby and Rails installed, then do the following :

1. bundle install
2. rake db:create
3. rake db:schema:load
4. bin/rails server

And visit to http://localhost:3000/
